# Segmentasi Pelanggan Toko Buku dengan RFM & K-Means

Proyek ini bertujuan untuk meningkatkan efektivitas pemasaran dan stabilitas pendapatan Toko Buku dengan cara memahami dan mengelompokkan basis pelanggan. Sebagai Data Scientist, proses dimulai dengan membersihkan dan memvalidasi data transaksi historis dari tahun 2010 hingga 2024, memastikan tidak ada nilai yang hilang atau duplikat yang mengganggu. Data yang telah dibersihkan kemudian diubah melalui Feature Engineering dari transaksi mentah menjadi tiga metrik kunci yang berharga secara bisnis: Recency, Frequency, dan Monetary (RFM). Untuk mempersiapkan data agar dapat diproses model secara efektif, kami menerapkan Log Transformation untuk mengatasi outlier dan Standard Scaling untuk menyeimbangkan skala fitur. Setelah divalidasi menggunakan Metode Siku (Elbow Method), model K-Means Clustering dilatih dan berhasil mengelompokkan pelanggan ke dalam empat segmen yang berbeda. Temuan kunci dari segmentasi ini adalah identifikasi kelompok Pelanggan VIP/Champion dan Pelanggan Berisiko Churn. Hasil ini memungkinkan tim Marketing untuk membuat keputusan yang sangat terfokus, misalnya, dengan menargetkan Kluster Berisiko Churn dengan kampanye retensi diskon. Sebagai deliverable akhir, kami menyimpan model K-Means yang telah divalidasi bersama dengan StandardScaler yang digunakan ke dalam file .pkl yang siap diserahkan kepada ML Engineer (A) untuk deployment sebagai API prediksi real-time.
